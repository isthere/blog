euphony.js v.0.2.0 release 및 통신 모듈레이션 방식의 중요성

Posted Oct 9, 2019

음파 통신 라이브러리인 euphony를 소개합니다.
euphony는 사람의 귀에 들리지 않는 가청주파수 대역에서도 높은 대역을 활용하여 스피커와 마이크만 가지고도 데이터 통신을 할 수 있는 사용성이 매우 편리한 라이브러리입니다. euphony는 크게 안드로이드와 웹 라이브러리 2가지 형태로 제공됩니다.

  • 라이브러리 소개 : https://dev.jbear.co/euphony
  • github repository : https://github.com/designe/euphony, https://github.com/designe/euphony.js

2013년도에 등장한 세계 최초의 음파통신 라이브러리이기도 합니다. 약 6년간 개발을 멈추고 있다보니 늦게 등장한 다른 상용화 솔루션 등에 속도나 안정성 측면에서 밀릴 수도 있지만 최근에 빠른 속도로 업데이트가 진행 중이며, 최종적으로는 많은 사람들이 쉽게 쓸 수 있는 편리하게 적용할 수 있는 가장 빠른 음파통신 라이브러리로써 성장하는 것이 목표입니다.

아래는 최근 업데이트된 패치 내역입니다. 관심 있는 분들은 언제든지 pull request 참여 부탁드리겠습니다.
