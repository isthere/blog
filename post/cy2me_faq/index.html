cy2me 싸이월드 백업 FAQ

Posted Jun 9, 2020

cy2me를 통해 싸이월드 백업하시면서
많은 분들이 q&a 에 질문을 남겨주셨어요.

CY2ME : https://github.com/designe/cy2me Q&A : https://blog.jbear.co/post/cyworld_backup/

궁금하신 부분들에 대한 답변을 정리해보았습니다.

싸이월드 접속 FAQ

  • 브라우저는 꼭 Chrome으로 해야하나요?
    Google Chrome, Microsoft Edge, Firefox가 가장 최신의 웹표준을 빠르게 반영하고 있습니다.
    Internet Explorer는 정상동작하지 않으니 위 3개의 브라우저를 이용해 백업해주시면 되겠습니다.
싸이월드 로그인이 안됩니다.

현재 서버가 매우매우 불안정합니다.
로그인을 하더라도 싸이월드 메인 페이지에서 미니홈피로 넘어가지 않는 경우가 있어요.
그리고 마치 로그인이 실패한 것처럼 뜹니다만, 실제로는 실패하지 않은 것입니다.
로그인 페이지에서 로그인 이후에 다른사람 홈피를 눌러서 자신의 미니홈피에 접근이 가능합니다.

싸이월드 미니홈피에 접속하니 An error occurred 가 뜹니다.

이 또한 서버가 매우 불안정하기 때문에 접속이 잘될때도 있고 잘되지 않을때도 있습니다.
현재 접속이 안된다고 해서 아예 안되는 것은 아니기 때문에 기다렸다가 다시 시도합니다.

크롬 개발 도구 스크립트 FAQ

댓글 백업도 가능한가요?

게시판, 사진첩, 다이어리 모두 댓글 백업이 기본적으로 활성화되어 있습니다.
댓글 데이터까지 모두 다운로드하다보니 속도가 조금 느린점 양해해주세요 :)

2010년 이전의 글은 백업이 되지 않나요?

많은 분들이 질문을 주셨는데 2009년 이전의 글도 문제없이 백업이 됩니다.
어떤 사용자들은 한번에 깔끔하게 백업을 성공하기도 하고 어떤분들은 듬성듬성 백업이 되는데 이는 서버 과부하 이슈이며 사용자가 많이 사용하지 않는 시간대에 다시 한번 해보시면 깔끔하게 백업을 성공하실 수 있습니다.

종종 502, 504 (bad gateway) 에러가 납니다.

본 백업 방식은 사용자가 페이지 한장 한장 접속해가며 백업하는 방식을 자동화한 결과물입니다.
따라서 시도할때 502, 504 에러는 현재 싸이월드 서버 상태가 매우 불안정함을 의미합니다.
사용자들이 많이 없는 시간대에 다시 시도해보시면 좋은 백업 결과를 받으실 수 있을 것입니다.

CY2ME 뷰어 FAQ

cy2me 뷰어를 통째로 받을 수 없나요?

매우 간단합니다. cy2me에 백업파일을 로드하신 후
오른쪽 클릭 -> 다른이름으로 저장 하시면 전체 파일이 통쨰로 저장됩니다.
또 인쇄 -> pdf로 출력하시면 pdf로 보관하실 수 있습니다 :)

Powershell FAQ

‘Invoke-WebRequest’ 용어가 cmdlet, 함수, 스크립트 파일 또는 실행할 수 있는 …
인수를 허용하는 위치 매개 변수를 찾을 수 없습니다.

파워쉘 버전 3.0 미만을 용용했을때 발생하는 문제입니다.
파워쉘 버전을 3.0 이상으로 업데이트 후 다시 해보시면 정상 백업이 가능합니다. https://www.microsoft.com/en-us/download/details.aspx?id=34595

저 또한 만족할만한 백업 솔루션이 없어서 제 입맛에 맞추어 만든 솔루션이다보니
모든분들께 만족을 드리기는 어려울 것 같아요. 아무쪼록 모두 깔끔하게 백업을 성공하셔서
